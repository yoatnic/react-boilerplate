# react-boilerplate

This is boilerplate for creating SPA using with React + Redux.

## Libs

- React
- React-Router
- Redux
- TypeScript
- Webpack4
- Jest
- tslint
- prettier

## Getting start

### install Node.js

https://nodejs.org/en/

Required version >=6.11.5

### Install dependencies

Installing depended modules of this project.
Type following commands on your terminal.

```sh
npm install
```

Or if you use yarn.

```sh
yarn install
```

### Run dev server

Type following commands on your terminal.

```sh
npm start
```

### Test

```sh
npm test
```

### Lint

```sh
npm run lint
```

Lint target is `/src`

### Type checking

```sh
npm run flow
```

### build

```
npm run build
```

After, run any http server at `/public`
